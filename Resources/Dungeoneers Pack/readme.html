Description

Doodads and destructibles for designing your own, bigger, better dungeons. Currently includes a full set of mine walls, an almost completed set of stone walls, and some other stuff. More will come in the future.

Pathing Textures:
Most walls are made to fit with either of the following pathing textures, for easy placement in the map:
PathTextures\4x2Default.tga
PathTextures\2x2Default.tga

If not, the correct pathing texture will be in the readme in specific file.
