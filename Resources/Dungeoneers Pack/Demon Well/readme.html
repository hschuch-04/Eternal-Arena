Description

Inspired by the Blood Fountain shrine from DIABLO (The original), which I for some reason still remember.

Comes in two variations: Health Fountain and Mana Fountain, with 4 animations each to simulate them being used. Each also comes with an alternate animation where the fountain has been destroyed. The death and birth animation was mostly added to widen the models usage, as it is meant to be a neutral building, not a buildable structure.

Uses only ingame textures.
