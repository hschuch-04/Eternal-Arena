Description

The main function of this model is the fact that it contains 5 differently placed attachment points inside of it, with which you can attach whatever your heart desires. Treasure models, units, random junk, it all works. This is supposed to be mainly useful for RPG's, or anything where you complete a quest or find a chest in a dungeon.

This is also fully animated to include:

* Stand
* Stand Alternate
* Morph (Opens)
* Morph Alternate (Closes)
* Death
* Death Alternate
* Birth

Attachment points as follows:

* Back left
* Back right
* Origin
* Front left
* Front Right
* Keyhole

The keyhole attachment rotates, so that when a key model is attached, the key will turn and unlock the chest in the opening animation, or lock it in the closing animation.
