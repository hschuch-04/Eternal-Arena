Description

Hi guys, those are some module parts of HD war3 that I found pretty useful for environment artists. I have been working on Quenching Mod with my friends for a year, and I have a small collection of helpful prop models that I want to share. Those are just simply parts of the existing blizzard models, but I found them extremely helpful.

There is a simple guide on how to use them.

First, thanks a lot for ReterasModelStudio which saves me tons of time to make those models.

Second, those are for terrain arts only, it's pretty hard to use them for actual building cause they don't have real animation.

Third, you don't need any extra textures, all of them used default textures.

Fourth, better to use click helper for those modules.
